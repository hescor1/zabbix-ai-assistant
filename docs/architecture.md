\# Architecture



\## Project Vision



The goal of this project is to build an AI assistant for Zabbix.



The assistant should be able to:



\* Query the Zabbix API

\* Read technical documentation

\* Search operational knowledge

\* Work with local and cloud AI models



\## Environment 1 - Home Lab



Features:



\* Internet access

\* OpenAI support

\* Ollama support

\* Experimentation environment



Purpose:



\* Learning

\* Development

\* Testing



\## Environment 2 - Enterprise



Features:



\* Restricted Internet access

\* Local Ollama models

\* Local documentation

\* Zabbix API access



Purpose:



\* Security

\* Compliance

\* Operational support
